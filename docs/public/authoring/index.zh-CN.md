# Cap4k 编写指南总览

[English](index.md)

> 这套文档定义 cap4k 项目的默认编写方式、审计方式，以及偏离默认路径时的决策入口。

## 这套文档解决什么问题

- 让项目作者知道如何直接开始编写 cap4k 项目
- 让审阅者知道如何按 Default Happy Path 审核项目
- 为后续 AI collaboration guide 提供稳定上游规则
